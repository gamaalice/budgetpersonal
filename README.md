# Personal Budget Manager 
A simple and organized personal budgeting application built with pure Tkinter, Pandas, and Matplotlib.
Data is stored in a clean CSV file that starts empty, and all transactions are added through the interface.

- Rightâ€‘side input panel (date, category, subcategory, amount)
- Predefined financial categories and subcategories
- Automatic CSV creation with header only
- Transactions table with delete option
- Pie chart: spending distribution
- Bar chart: total by category
- Clean dashboard layout
- Fully modular code (main.py, utils.py, charts.py)

## How to Run
pip install pandas matplotlib
python main.py
